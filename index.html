REST API

Метод URL Параметры Описание Ответ
GET /pair ?id_win=5 (опционально) Возвращает случайную пару (исключает id_win) JSON: массив из 2 объектов Person (с base64Photo)
GET /pair — Любая случайная пара JSON: массив из 2 объектов
POST /NewElo winnerId, loserId (form-data) Пересчитывает ELO после голосования —
GET /top — Топ всех участников JSON: массив всех Person, отсортированных по ELO DESC

Схема базы данных

CREATE TABLE person (
    id      BIGSERIAL PRIMARY KEY,
    name    TEXT NOT NULL,
    elo     INTEGER DEFAULT 1500 NOT NULL,
    photo   BYTEA                    -- фото хранятся как байты
);

При отдаче фото сразу конвертируются в base64 на бэкенде (поле base64Photo в JSON)
